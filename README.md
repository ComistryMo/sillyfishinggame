# 钓鱼：然后捡到女高中生

一个轻量的单文件钓鱼小游戏：点击开始，等鱼上钩，1 秒内按下空格就能收获。现在加入了“林汐”事件线——在河流或湖泊钓鱼时，有机会钓起漂流瓶，解锁一位被困浅滩的女高中生环保社团实习生。帮助她吃饱、安排救援，她会陪你一起钓鱼，缩短上钩等待时间。

## 玩法亮点
- 基础钓鱼：上钩提示后 1 秒内按空格判定成功。
- 多场景：家里切换“小溪 / 河流 / 湖泊”，稀有度和等待时间各不相同。
- 图鉴收集：统计每条鱼的捕获次数和最大重量。
- 林汐事件：
	- 在河流/湖泊钓鱼随机触发漂流瓶求救，解锁“事件”入口。
	- 每次成功钓鱼会自动分出一部分鱼肉给她，提升信任、积累补给。
	- 补给累计 ≥ 8kg 时，可在事件面板安排救援；救出后等待时间获得持续加成。
	- 可与林汐聊天小互动，缓慢提升信任。

## 操作说明
- 开始钓鱼：点击“开始钓鱼”。
- 捕获判定：看到“上钩！”后 1 秒内按空格。
- 取消钓鱼：点击“取消钓鱼”。
- 场景切换：在家选择“小溪 / 河流 / 湖泊”。
- 事件入口：当漂流瓶事件触发后，家中的“事件地点”会亮起“前往林汐的浅滩”。

## 运行方式
```bash
python ./fishing_game.py
```

依赖：Python 3，自带 tkinter（Windows 默认内置）。

## 存档
- fishing_stats.json 会记录鱼类统计、金币（预留）、以及林汐事件进度。

## 小贴士
- 想快点遇到漂流瓶，多去河流或湖泊钓几次。
- 救出林汐后，信任越高，等待时间越短（下限 0.6 倍）。
